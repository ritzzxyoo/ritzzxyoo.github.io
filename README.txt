Ritika's Portfolio — Local Setup
================================

Files:
  index.html   - main page
  style.css    - styles
  script.js    - interactions
  avatar.png   - hero avatar image

How to run:
  Option 1 (easiest): just double-click index.html to open it in your browser.

  Option 2 (recommended, for best results):
    Open a terminal in this folder and run any one of these:
      python -m http.server 5500
      npx serve .
    Then visit http://localhost:5500 in your browser.

Notes:
  - The "Download Resume" button looks for resume.pdf in this folder.
    Drop your real resume.pdf here to enable it.
  - Project GitHub/Live links and LinkedIn/GitHub icons in Contact are
    placeholders (#) — replace them in index.html with your real URLs.
